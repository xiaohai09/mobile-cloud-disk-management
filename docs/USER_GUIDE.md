# Caiyun Refactored - 使用说明

## 用户角色

| 角色 | 说明 |
|------|------|
| 管理员 | 可管理用户、账号、任务配置、系统配置 |
| 普通用户 | 可管理自己的账号、查看任务日志、兑换记录 |

## 首次登录

1. 访问 http://localhost
2. 使用管理员账号登录
3. 默认管理员账号在初始化数据库时创建

## 功能模块

### 账号管理

- 添加云盘账号（手机号 + 短信验证码登录）
- 批量导入账号
- 查看账号状态、过期时间、云朵数
- 手动触发单个账号任务

### 任务管理

- 查看任务日志
- 批量触发任务
- 查看队列状态

### 兑换中心

- 搜索可兑换商品
- 自动抢兑配置
- 查看抢兑记录

### 数据导出

- 导出账号列表为 CSV/JSON
- 导出任务日志
- 导出兑换记录
- 历史导出记录管理

### Webhook 通知

- 创建 Webhook 端点
- 订阅事件类型：`task.success`、`task.failed` 等
- 配置 HMAC 签名密钥
- 测试端点连通性
- 查看投递日志

### 移动端使用

- 响应式布局自动适配手机屏幕
- 底部导航栏快速切换模块
- 支持暗色模式（跟随系统或手动切换）

## 常见问题

### 忘记密码

点击登录页“忘记密码”，输入注册手机号获取重置码。

### 账号被封禁

请检查：
1. 是否频繁触发任务
2. JWT 获取失败次数是否过多
3. 是否在合规时间段内操作

### 导出失败

- 检查数据量是否过大
- 确认磁盘空间充足
- 查看后台任务日志

### Webhook 收不到通知

- 确认 URL 可达且支持 POST
- 检查事件订阅列表是否包含对应事件
- 查看投递日志中的错误信息
- 确认签名密钥配置正确
