# 用户指南

## 用户角色

系统支持三种用户角色：

| 角色 | 权限范围 |
|------|----------|
| **普通用户** | 查看自己的账号、任务、兑换记录 |
| **管理员** | 管理所有用户、任务、兑换配置、公告 |
| **超级管理员** | 系统配置、审计日志、所有管理员权限 |

## 首次登录

1. 访问系统首页
2. 使用默认管理员账号登录：
   - 用户名：`admin`
   - 密码：首次登录后需立即修改
3. 登录后进入仪表盘，查看系统概览

## 功能模块

### 仪表盘

- 系统健康状态总览
- 任务队列监控
- 兑换成功率统计
- 公告通知中心

### 账号管理

- 添加/编辑/删除云盘账号
- 批量导入账号
- 账号状态监控（在线/离线/异常）
- 自动签到状态查看

### 任务中心

- 创建抢兑任务
- 查看任务执行日志
- 任务启停控制
- 任务执行结果统计

### 兑换中心

- 商品浏览与筛选
- 创建兑换任务
- 兑换记录查询
- 兑换结果导出

### 导出中心

- 支持导出兑换记录、任务日志、账号信息
- 导出格式：CSV、JSON
- 按时间范围、账号筛选
- 导出历史记录管理

### Webhook 管理

- 创建 Webhook 端点
- 订阅事件类型
- 查看投递日志
- HMAC 签名验证配置

### 系统设置

- 修改个人信息
- 更换头像
- 主题切换（亮色/暗色/自动）
- 密码修改

## 常见问题

### 1. 账号添加失败

**可能原因**：
- 账号密码错误
- 网络连接超时
- 云盘服务端限流

**解决方法**：
- 检查账号密码是否正确
- 稍后重试
- 查看任务日志获取详细错误信息

### 2. 兑换任务未执行

**可能原因**：
- 任务未启用
- 到达执行时间但队列繁忙
- 账号 Cookie 过期

**解决方法**：
- 检查任务状态是否为“启用中”
- 查看队列监控页面
- 更新账号 Cookie

### 3. 导出文件下载失败

**可能原因**：
- 导出任务未完成
- 文件已过期（7天自动清理）
- 磁盘空间不足

**解决方法**：
- 等待任务完成后再下载
- 重新创建导出任务
- 联系管理员清理磁盘

### 4. Webhook 接收失败

**可能原因**：
- 目标 URL 无法访问
- HMAC 签名验证失败
- 响应状态码非 2xx

**解决方法**：
- 检查目标服务是否可访问
- 验证 Webhook secret 配置
- 查看投递日志中的详细错误信息

## 技术支持

如遇到其他问题，请：

1. 查看系统日志：`docker compose logs -f`
2. 检查 Grafana 监控面板
3. 联系系统管理员
