# 国际化文档说明

本项目文档采用中英双语，默认展示中文。

## 文档索引

- `docs/INDEX.md` — 中文文档中心（默认）
- `docs/INDEX.en.md` — English Documentation Center

## 文档列表

| 中文（默认） | English |
|-------------|---------|
| `README.md` | `README.en.md` |
| `docs/INDEX.md` | `docs/INDEX.en.md` |
| `docs/DEPLOYMENT.md` | `docs/DEPLOYMENT.en.md` |
| `docs/API.md` | `docs/API.en.md` |
| `docs/USER_GUIDE.md` | `docs/USER_GUIDE.en.md` |
| `docs/TEST_REPORT.md` | `docs/TEST_REPORT.en.md` |
| `CONTRIBUTING.md` | `CONTRIBUTING.en.md` |

## 语言切换说明

- **默认展示语言**：中文
- **切换方式**：阅读对应 `.en.md` 后缀的英文文档
- **文档中心**：从 `docs/INDEX.md` 进入，可按语言切换

## 多语言维护规范

1. 新增文档时，必须同时创建中英文两个版本
2. 中文文档使用 `.md`，英文文档使用 `.en.md`
3. 修改中文文档后，必须同步更新对应的英文版本
4. 提交 PR 时，需确认中英文内容一致性
