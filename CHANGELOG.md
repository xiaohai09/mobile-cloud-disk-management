# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-06

### Added
- Docker-only deployment with multi-stage builds
- Go backend with Clean Architecture (domain/application/infrastructure/interfaces)
- Vue3 responsive frontend with Pinia state management
- Data export center with multiple formats (Excel, CSV, JSON, PDF)
- Webhook notification system with retry mechanism
- Platform adapter for multi-platform deployment
- Real-time WebSocket communication
- Prometheus metrics and Grafana dashboards
- Playwright E2E test suite (10/10 passed)
- Bilingual documentation (Chinese default + English)
- Frontend i18n with Chinese/English switching
- GitHub Actions CI/CD workflows
- GitHub Pages documentation site
- Automated release workflow on tag push
- Stale issues auto-close policy
- Security policy and vulnerability reporting
- Contributing guide with development standards
- Branch protection and CODEOWNERS
- Dependabot configuration for dependency updates

### Changed
- Project renamed to "移动云盘管理系统" (Mobile Cloud Disk Management System)
- All documentation synchronized in Chinese and English
- Repository topics updated to 20+ relevant tags
- GitHub repository description and homepage updated

### Security
- Enabled GitHub Advanced Security features
- Secret scanning alerts enabled
- Vulnerability alerts enabled
- Security policy published

### Documentation
- Added complete bilingual documentation set
- Created documentation index with navigation
- Added development environment setup guide
- Added API documentation with authentication flow
- Added user guide with role-based instructions
- Added test report with verification results
